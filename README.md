# bootstrap

Here you find some usefull I need to bootstrap my developer machine.

## VSCODE Shortcuts

* F7 - Build (cmake extensions)
* Shift+F5 - Debug (cmake extensions)
* CTRL-K - Line comment toggle
* CTRL-B - Enable/disable sidebar
